# Review

Ревью существующего opencode skill.

Прочитай все файлы скилла. Проверь по чеклисту.

## Checklist

### Frontmatter
- [ ] name — короткий, kebab-case
- [ ] description — понятно когда триггериться

### Router (если есть)
- [ ] Интенты покрывают все вероятные запросы
- [ ] Интенты не перекрываются (каждый запрос → ровно один интент)
- [ ] Модули загружаются минимально (не больше чем нужно)
- [ ] Write-таски грузят инварианты/безопасность

### Content
- [ ] Нет filler-фраз (just, simply, basically, of course)
- [ ] Нет повторов между модулями
- [ ] Каждая секция оправдывает своё существование
- [ ] Технические термины точные
- [ ] Примеры корректные

### Correctness
- [ ] Факты совпадают с реальной конфигурацией/репо
- [ ] Нет устаревших ссылок
- [ ] Нет противоречий между модулями

### Completeness
- [ ] Все важные правила из AGENTS.md (если были) переехали
- [ ] Edge cases учтены
- [ ] Запреты явно прописаны

### Usability
- [ ] Модель сразу поймёт что делать
- [ ] Инструкции однозначные
- [ ] Путь к файлам модулей указан (modules/...)

## Output
