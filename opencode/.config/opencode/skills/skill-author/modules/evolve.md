# Evolve

Добавление нового роута/модуля в существующий skill.

## Workflow

1. Определить новый интент
   - Когда он нужен (условие)
   - Какие модули загружает
   - Не пересекается с существующими

2. Обновить роутер в `SKILL.md`
   - Добавить строку в таблицу Intents
   - Если новый интент меняет описание скилла — обновить frontmatter description

3. Создать модуль в `modules/<name>.md`
   - По правилам из `modules/create.md`

4. Проверить консистентность
   - Старые модули не конфликтуют с новым
   - Базовые правила в SKILL.md актуальны
   - Описание скилла покрывает новый интент
