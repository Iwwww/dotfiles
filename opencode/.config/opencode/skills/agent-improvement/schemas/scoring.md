# Scoring

Use scores only as summaries, not as the main output.

## Process Quality

10: focused, efficient, validated, minimal context waste  
8: good, minor inefficiencies  
6: solved or partially solved, but with waste or weak validation  
4: significant drift, poor sequencing, poor validation  
2: mostly ineffective  
0: solved the wrong problem or caused major damage

## Context Hygiene

10: minimal reads, strong synthesis, no stale assumptions  
8: small context waste  
6: some repeated reads or missing synthesis  
4: significant context pollution or forgotten constraints  
2: context mostly uncontrolled  
0: context collapse

## Delegation Quality

10: subagents used only when high-leverage  
8: mostly good delegation  
6: some overuse or underuse  
4: delegation caused duplication or confusion  
2: harmful delegation  
0: no usable delegation model

## Validation Quality

10: strong relevant validation and diff inspection  
8: good validation with small gaps  
6: partial validation  
4: weak validation  
2: no meaningful validation  
0: false confidence despite no validation
